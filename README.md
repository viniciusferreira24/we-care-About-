# 🐾 We Care About

Landing page desenvolvida com foco em prática de estruturação e estilização utilizando HTML e CSS.

---

## 📌 Sobre o Projeto

O **We Care About** é uma página responsiva voltada para apresentação de serviços relacionados a cuidados com pets.

O projeto foi desenvolvido com o objetivo de praticar:

- Estruturação semântica com HTML5  
- Estilização moderna com CSS3  
- Organização de layout  
- Responsividade com Media Queries  

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## ⚙️ Funcionalidades

- Layout moderno e limpo  
- Estrutura semântica organizada  
- Design responsivo para diferentes tamanhos de tela  
- Botões estilizados  
- Uso de fontes externas (Google Fonts)
